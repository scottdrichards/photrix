8-Week Strength & Conditioning Program

DAY 1 (MON): UPPER BODY

Supplements (Optional): Multivitamin • Protein powder • Creatine 3–5 g/day
Exercise Sets Reps Notes / Equipment Set 1 Set 2 Set 3 Set 4
Dumbbell Bench Press 3 8–10 DBs; start 25–35 lb
One-Arm Dumbbell Row (bench-supported) 3 8–12/side 30–40 lb
Seated Dumbbell Shoulder Press 3 8–10 Neutral grip; 20–30 lb
Lateral Raise 3 12–15 10–15 lb
Close-Grip Floor Press 3 10–12 DBs; wrist-friendly
Optional Cardio (1–2x/week): Zone 2: 20–30 min (brisk walk, incline treadmill, easy jog)

Program Notes

  • Same program repeated for all 8 weeks
  • Progress by adding reps, time, or small weight increases
  • Stop any exercise that causes pain or discomfort

8-Week Strength & Conditioning Program

DAY 2 (TUE): LOWER BODY + CORE

Supplements (Optional): Multivitamin • Protein powder • Creatine 3–5 g/day

Lower Body

Exercise Sets Reps Notes / Equipment Set 1 Set 2 Set 3 Set 4
Goblet Squat 4 6–10 KB or DB at chest; ~35 lb
Romanian Deadlift 3 6–10 DBs; 35–45 lb
Reverse Lunge 3 8–10/leg DBs; 20–25 lb
Kettlebell Swing (SIT) 3 20s / 10s 25–30 lb; work/rest

Core Block

Exercise Sets Reps / Time Notes
Dead Bug 3 6–8/side Slow, controlled
Heel Slides 2–3 6–8/side Pelvis still
Bird Dog 2–3 6–8/side 2s pause
Carry (Suitcase OR Front) 2–3 20–30 sec 25 lb KB

Core Notes

  • Stop any core work that causes coning, pelvic pressure, or back pain
  • Focus on breathing and maintaining neutral spine
